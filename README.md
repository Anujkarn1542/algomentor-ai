# AlgoMentor AI

AI-powered DSA learning platform that helps students solve coding problems with:
- AI hints
- step-by-step dry run
- code explanation
- complexity analysis
- interview tips

## Tech Stack
- React
- TypeScript
- Tailwind
- Node.js
- Express
- MongoDB
- OpenAI API / Gemini API

## Features
- AI code explanation
- problem solving mentor
- dark mode
- chat interface
- code dry run visualization
